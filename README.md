# Todo List App

## Overview

This project is part of The Odin Project curriculum. It focuses on building a dynamic Todo List application using JavaScript modules and Webpack.

The goal was to practice structuring code, managing application state, and working with browser storage.

---

## Features

* Create new projects
* Add todos to projects
* View all projects and their todos
* Delete todos
* Data persistence using localStorage

---

## Current Limitations

* Expand/edit todo functionality not implemented yet
* No CSS styling (functionality-focused)

---

## Technologies Used

* JavaScript (ES6 Modules)
* Webpack
* LocalStorage
* HTML

---

## What I Learned

* Organizing code using modules
* Managing data and UI separately
* Handling DOM manipulation dynamically
* Using localStorage to persist data
* Working with Webpack (development vs production builds)

---

## Future Improvements

* Add edit/expand functionality for todos
* Improve UI/UX with CSS
* Refactor code for better scalability

---

## How to Run

```bash
npm install
npm run start
```

To build for production:

```bash
npm run build
```
